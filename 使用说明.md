# 瞌睡虫V5趋势分析软件 - 使用说明

## 一、软件概述

本软件是一款专业的彩票3D号码趋势分析工具，通过多维度数据分析和AI算法，帮助用户进行号码筛选和预测。主要功能包括：自动分割匹配、后台智能分割、AI智能分析、评分权重配置、模式生成推荐、K线图表展示等。

**技术特点**：
- 10大评分指标综合评估
- K线形态识别与贝叶斯平滑
- 多模型集成投票系统
- 模式库自学习与进化
- 后台多线程智能分割
- 三重数据持久化备份

**两步分割流程**：
1. **第一次分割**：根据选择的趋势图模板（如8中7），从1000个号码中搜索符合条件的号码组
2. **第二次分割**：使用第一次分割的结果作为号码池，应用第二个模板（如4中3），进一步筛选

**功能逻辑**：分割结果评分低于阈值时自动重试，直到满足条件或达到最大次数

### 2.4 后台智能分割 ⚡

**位置**：自动分割匹配下方

**核心功能**：后台自动执行8中7分割，寻找5连挂最优号码组

#### 2.4.1 开启条件

点击开关开启，系统会立即执行一次分割循环（需至少6期开奖数据）

#### 2.4.2 工作流程（三步循环）

**第1步：搜索4连挂号码组**
- 使用模板 `0000`（连续4期未中）
- 使用**上上期**开奖号判断（如即将开50期，用48期判断）
- 目标：搜索并保存100组恰好4连挂的号码组

**第2步：过滤5连挂号码组**
- 使用**上期**开奖号检查（如即将开50期，用49期判断）
- 筛选出在上期仍未命中的号码组（即5连挂）
- **严格过滤**：只保留恰好5连挂的组，6连挂及以上全部淘汰

**第3步：评分选最优**
- 使用10维度评分机制对5连挂组进行评分
- 选取评分最高的一组作为最优组
- 显示在卡片中，提供复制功能

#### 2.4.3 持续循环

- 每次开奖数据更新后，自动重新执行完整循环
- 如果第一次分割选择了「⚡ 使用后台分割号码」，自动分割会等待后台分割完成后再执行
- 多线程执行，不阻塞主界面

#### 2.5.2 模式生成655注（方案D：组采样法）

**生成逻辑（三层筛选）**：

**第一层：6个单号码有效指标评分**
- 对 000-999 共1000个号码分别评分
- 使用指标：历史命中率、回归期望、最大连错、趋势、K线形态、多模型投票
- 按各自权重加权求和 → 单号码得分

**第二层：取候选池前800注**
- 按单号码得分降序排序
- 取前800注作为候选池（留出145注优化空间）

**第三层：组采样法选最优655注**
- 从候选池随机采样655注，重复M=200次
- 每次用4个组指标评估：冷热平衡、多样性、位置转移、奇偶大小比
- 综合得分 = 单号码平均分 × 60% + 组指标得分 × 40%
- 200次采样后，选综合得分最高的655注

**统计内容**：

| 统计项 | 说明 |
|--------|------|
| 生成次数 | 模式生成655注的总次数 |
| 命中次数 | 生成号码命中开奖号的次数 |
| 未命中 | 生成号码未命中开奖号的次数 |
| 命中率 | 命中次数 / (命中+未命中) |
| 连中 | 当前连续命中次数 |
| 连挂 | 当前连续未命中次数 |

**操作按钮**：
- 🗑️ 清除统计：清空所有统计数据
- 📋 复制号码：复制生成的655注号码

**防重复验证机制**：
- 记录已验证的期号，避免重复统计
- 只验证与生成目标期匹配的开奖结果
- 防止因多次刷新导致命中率异常

#### 2.5.3 评分系统（10维度）

**10大评分指标详解**：

| 指标 | 默认权重 | 类型 | 说明 |
|------|----------|------|------|
| 多模型投票 | 24 | 核心 | 多个独立模型一致推荐的比例，集成学习降低单模型偏差 |
| 模式评分 | 24 | 核心 | 与模式库中历史高命中模式的匹配程度 |
| K线形态 | 24 | 核心 | 基于8种K线形态的历史命中率统计（贝叶斯平滑+多周期融合） |
| 回归期望值 | 5 | 重要 | 基于连挂状态的回归概率，连挂越多回归期望值越高 |
| 趋势评分 | 2 | 重要 | 趋势通道模型（布林带+EMA+MACD+KDJ）分析结果 |
| 冷热平衡度 | 5 | 辅助 | 热号/温号/冷号的比例均衡度，越均衡评分越高 |
| 最大连错 | 5 | 辅助 | 历史最大连续未中的期数，连错越少评分越高 |
| 位置转移 | 5 | 辅助 | 号码位置分布转移概率的合理性 |
| 平均遗漏值 | 1 | 辅助 | 号码组平均遗漏期数，冷号组适当加分 |
| 奇偶大小比 | 5 | 辅助 | 奇偶比和大小比是否接近理论均衡值 |

**评分公式**：
```
综合得分 = Σ(各指标得分 × 指标权重) / 总权重
```

**K线形态指标详解**：

8种核心形态：
| 形态 | 说明 |
|------|------|
| hot（热期） | 近N期命中率>65% |
| cold（冷期） | 近N期命中率<35% |
| sideways（横盘） | 近N期命中率35-65% |
| streak_up（连红） | 连续2-3根红柱 |
| streak_down（连绿） | 连续2-3根绿柱 |
| alternating（交替） | 近4期红绿交替 |
| v_bottom（V型底） | 连绿2期后转红 |
| inv_v_top（倒V顶） | 连红2期后转绿 |

多周期融合：
- 5期周期：权重 50%（近期形态）
- 10期周期：权重 35%（中期形态）
- 20期周期：权重 15%（长期形态）

贝叶斯平滑：先验概率27.1% + 实际观测，解决小样本过拟合

#### 2.5.4 评分权重配置

## 三、使用流程

### 3.1 基础使用流程

1. **导入数据**：确保系统已导入足够的历史开奖数据（至少830期）
2. **创建趋势图**：在K线编辑器中绘制分割模板
3. **手动分割**：点击「开始智能分割」执行首次分割
4. **查看结果**：右侧K线图显示分割结果，排名徽章标注优劣
5. **开启自动分割**：首次分割完成后，开启「开奖自动分割」开关

### 3.2 后台智能分割使用流程

1. **开启后台分割**：点击「后台智能分割」开关
2. **等待搜索完成**：观察进度条，等待5连挂最优组生成
3. **设置第一次分割**：在分割配置中选择「⚡ 使用后台分割号码」
4. **开启自动分割**：开启后，每次开奖更新自动使用最新最优组

### 3.3 AI分析使用流程

1. **配置权重**：在评分权重设置中调整各指标权重（可选）
2. **执行分割**：完成智能分割后自动触发AI分析
3. **查看推荐**：在AI分析报告中查看评分最高的推荐号码
4. **复制号码**：点击「复制号码」按钮获取推荐号码组

### 3.4 模式生成655注使用流程

1. **确保数据充足**：至少需要30期历史开奖数据
2. **配置权重**：调整10个指标的权重配置
3. **生成号码**：点击「🔮 模式生成655注」或等待自动生成
4. **查看统计**：观察命中率统计，验证效果
5. **复制号码**：点击「📋 复制号码」获取655注号码

### 3.5 模式库训练流程

1. **开始训练**：点击「🚀 开始训练」按钮
2. **等待训练**：模式数量逐步增加至10000条
3. **回溯验证**：定期执行回溯验证，淘汰低质量模式
4. **导出备份**：定期导出模式库JSON备份

## 五、注意事项

### 5.1 数据要求

1. **最少数据**：至少需要6期开奖数据才能使用后台智能分割
2. **推荐数据量**：建议830期以上，评分更准确
3. **4连挂概率**：每组8中7分割的4连挂概率约为1/4096，搜索可能需要较长时间
4. **数据格式**：系统使用开奖号后3位进行分析，确保数据格式正确

### 5.2 评分系统

1. **评分阈值**：总分低于20分的推荐不会显示
2. **权重影响**：权重配置直接影响评分结果和推荐排序
3. **K线形态**：需要足够历史数据才能准确统计
4. **贝叶斯平滑**：样本量不足时评分趋近理论值，需积累数据

### 5.3 模式生成

1. **随机性**：组采样法含随机因素，每次生成结果可能不同
2. **采样次数**：M=200次平衡性能与质量，可根据需要调整
3. **候选池**：800注候选池保证质量的同时留出优化空间
4. **统计准确性**：需积累足够多的生成次数，命中率才有参考价值

### 5.4 后台分割

1. **优先级**：当第一次分割选择「使用后台分割号码」时，自动分割会等待后台分割完成
2. **多线程**：后台分割在Web Worker中运行，不阻塞主界面
3. **循环机制**：每次开奖更新后自动重新执行

## 七、常见问题

### Q1：为什么后台分割搜索很慢？
A：8中7的4连挂概率约为1/4096，需要搜索大量组合。请耐心等待，搜索在后台进行不影响其他操作。

### Q2：为什么AI分析没有推荐结果？
A：可能所有组的评分都低于20分阈值。可以降低评分阈值，或检查数据是否充足。

### Q3：模式生成的命中率不准怎么办？
A：需要积累足够多的生成次数（建议>30次），命中率统计才有参考价值。初始阶段可能波动较大。

### Q4：K线形态评分为什么总是接近50分？
A：样本量不足时，贝叶斯平滑会使评分趋近理论值（27.1%映射后约50分）。积累更多历史数据后评分会更准确。

### Q5：如何提高推荐命中率？
A：
1. 增加历史数据量
2. 合理配置权重，适合自己的玩法
3. 定期训练和优化模式库
4. 结合多种指标综合判断
5. 不要盲目跟单，理性购彩

### Q6：数据会丢失吗？
A：系统采用三重备份机制（localStorage + 后端 + IndexedDB），数据安全性较高。建议定期导出模式库做额外备份。
